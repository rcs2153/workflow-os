# Maintainers

Workflow OS maintainers are responsible for project direction, review quality, release decisions, and security response.

## Current Maintainers

- Project maintainers to be named before the first public release.

## Maintainer Responsibilities

- Enforce [docs/ENGINEERING_STANDARD.md](docs/ENGINEERING_STANDARD.md).
- Preserve the product boundary in [docs/PROJECT_CHARTER.md](docs/PROJECT_CHARTER.md).
- Review security-sensitive changes carefully.
- Coordinate vulnerability disclosure.
- Keep release notes, ADRs, and public contracts accurate.
